TCP 服务器-客户端程序运行说明文档：
简介
本项目包含一个TCP服务器和一个TCP客户端。服务器接收客户端的连接请求，处理数据并将处理结果返回给客户端。客户端读取一个文件，将文件内容以数据块的形式发送给服务器，然后接收服务器返回的反转文本并将其保存到一个新文件中。

文件说明
server.py: 包含服务器端代码，负责监听客户端连接并处理数据。
client.py: 包含客户端代码，负责连接服务器并发送数据。

运行环境
Python
需要以下Python库：
argparse
random
socket
selectors
time


使用说明

服务器端
配置选项

--ip: 指定服务器的IP地址（默认值为 0.0.0.0）。
--port: 指定服务器的端口号（默认值为 12345）。

客户端
配置选项

Lmin: 数据块长度的最小值（必填）。
Lmax: 数据块长度的最大值（必填）。
server_ip: 服务器的IP地址（必填）。
server_port: 服务器的端口号（必填）。

准备输入文件
确保 ASCII.txt 文件存在，并包含要发送的数据。

注意事项
确保服务器先于客户端启动。
确保服务器和客户端运行在同一网络环境中，且防火墙允许相应的端口通信。
确保客户端的 ASCII.txt 文件存在并有足够的数据。
运行命令时，根据实际情况调整 Lmin 和 Lmax 参数。

测试步骤
启动服务器。
准备客户端的 ASCII.txt 文件。
启动客户端，连接到服务器。
检查服务器和客户端的输出。
确认 reversed.txt 文件包含反转后的文本数据。

常见问题
无法连接服务器：检查服务器是否已启动，IP地址和端口号是否正确。
数据传输异常：确保客户端发送的数据格式正确，服务器能够正确解析和处理。
文件读取异常：确保 ASCII.txt 文件存在且具有足够的数据
